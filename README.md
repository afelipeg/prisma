# Prism — Starter Kit

> One schema. Every surface. The framework that makes a business agent-native from line zero.

This kit gives you everything to start building **Prism** with Claude Code tomorrow.

## What's in the box
| File | Purpose |
|---|---|
| `CLAUDE.md` | The brain. Drop at repo root — Claude Code reads it every session. Full architecture, invariants, conventions, Day-1 DoD. |
| `bootstrap.sh` | Run once → scaffolds the entire TypeScript monorepo (5 packages + demo app + docs). |
| `DAY1_DEMO.md` | Hour-by-hour runbook with the exact prompts to paste into Claude Code, in order. |
| `product.prism.ts` (reference) | The schema DSL recommendation, fully worked for Café Nube. (Also generated by bootstrap into `apps/cafe-nube/prism/`.) |

## 60-second start
```bash
curl -fsSL https://bun.sh/install | bash      # if you don't have bun
bash bootstrap.sh prism
cd prism && bun install
cp ../CLAUDE.md ./CLAUDE.md                    # if not already copied in
cp ../DAY1_DEMO.md ./docs/DAY1_DEMO.md
claude                                         # open Claude Code, follow DAY1_DEMO.md
```

## The architecture in one breath
You write your business once in `*.prism.ts` using `definePrism()`. That produces a
typed **IR** (intermediate representation). Emitters turn the IR into four surfaces:

```
*.prism.ts → IR → ┬ Human UI      (Next.js + JSON-LD)
                  ├ Agent API     (MCP tools + OpenAPI)
                  ├ Knowledge     (JSON-LD, llms.txt, entity graph)
                  └ Monetization  (checkout + ARTF signals — stubbed for Day 1)
```

A future custom `.prism` DSL compiles **down to the same IR**, so emitters never change.

## Decisions baked in (and why)
- **TypeScript + Bun + Turborepo** — one language, one runtime; max velocity for Claude Code. Go (for ARTF) waits until Phase 3.
- **`.prism.ts` builder, not a custom parser** — 90% of the DSL magic, 10% of the effort, native to Claude Code. The IR is designed so a real `.prism` grammar can target it later.
- **In-memory store for the demo** — swappable for SQLite/Postgres behind `@prism/runtime` with zero emitter changes.
